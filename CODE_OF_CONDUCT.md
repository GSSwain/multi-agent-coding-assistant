# Code of Conduct

This project is intended to be a welcoming and respectful community.

## Our standards
- Be kind, respectful, and professional.
- Assume good faith in others.
- Avoid harassment, discrimination, or disruptive behavior.

## Enforcement
Project maintainers may remove, edit, or reject comments, issues, or pull requests that violate this code of conduct.

## Reporting
Please report unacceptable behavior to the project maintainers via the repository issue tracker or contact channel.
